Formatting In HTML

They are used to display special type of text.
Examples:
< strong > displays IMPORTANT TEXT (would look like "book" )
< b > displays BOLD TEXT (would look like "book" )
< i > displays ITALIC TEXT (would look like "book" )
< small > displays SMALLER TEXT (would look like "book" )
< em > displays EMPHASIZED TEXT (would look like "book" )
< mark >displays MARKED TEXT (would look like "book" )
< ins > displays INSERTED TEXT (would look like "book" )
< del > displays DELETED TEXT (would look like "book" )
< sub > displays SUBSCRIPT TEXT (would look like "book" )
< sup > displays SUPERSCRIPT TEXT (would look like "book" )
