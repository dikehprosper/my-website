Basics In HTML

1. All HTML documents must start with a document type declaration: < !DOCTYPE html >.
2. The HTML document itself begins with < html > and ends with < /html >.
3. The visible part of the HTML document is between < body > and < /body >.
4. The < !DOCTYPE > declaration represents the document type, and helps browsers to display web pages correctly.It must only appear once, at the top of the page (before any HTML tags).
5.The < !DOCTYPE > declaration is not case sensitive.
6.The < !DOCTYPE > declaration for HTML5 is: < !DOCTYPE html >
